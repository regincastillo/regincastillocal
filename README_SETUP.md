#INSTALLATION
1.Please provide and change this following (varibles/env) in `.env`
```
REACT_APP_API_KEY=<google-api-key>

REACT_APP_CALENDAR_ID=<google-calendar-id>

REACT_APP_CLIENT_ID=<google-client-id>

```
2.Run this script to install all packages needed
`npm install`

3.Run this script to run the app
`npm run start`

4.Run this script to run the unit test
`npm run test`
